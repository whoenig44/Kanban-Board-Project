# Kanban Board Application

This is a Kanban board application built with a React frontend and an Express backend. The application allows users to manage tasks using a Kanban board interface.

## Features

- User authentication (login and logout)
- JWT-based authentication
- Task management (create, read, update, delete tasks)
- Drag-and-drop functionality for tasks
- Responsive design

## Technologies Used

### Frontend

- React
- TypeScript
- Axios
- JWT-decode

### Backend

- Express
- TypeScript
- Mongoose (MongoDB)
- Bcrypt
- JSON Web Token (JWT)

## Getting Started

### Prerequisites

- Node.js
- npm or yarn
- MongoDB

### Installation

1. Clone the repository:

   ```bash
   git clone git@github.com:whoenig44/Kanban-Board-Project.git
   cd kanban-board
   ```
